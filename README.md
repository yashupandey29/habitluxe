# Habitluxe

A premium habit tracking web application built using Node.js, Express.js, MongoDB, HTML, CSS and JavaScript.

## Features
- User Authentication
- Habit Tracking
- Daily Streak System
- Progress Statistics
- Admin Dashboard
- Email Notifications
- Responsive Design

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Deployment: Render
